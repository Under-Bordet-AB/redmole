# Weather Icons LVGL Font

`weather_icons_128.c` is a generated LVGL font subset built from Erik Flowers'
Weather Icons TTF:

https://github.com/erikflowers/weather-icons

Included glyphs:

- `wi-day-sunny` `0xf00d`
- `wi-day-cloudy` `0xf002`
- `wi-cloudy` `0xf013`
- `wi-fog` `0xf014`
- `wi-sprinkle` `0xf01c`
- `wi-rain` `0xf019`
- `wi-snow` `0xf01b`
- `wi-thunderstorm` `0xf01e`
